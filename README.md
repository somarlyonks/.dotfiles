## Usage

Make sure in `bash`.

### clone

```bash
git clone git@github.com:somarlyonks/.dotfiles.git ~/.dotfiles
```

### checkout to new branch

```bash
cd ~/.dotfiles && git checkout -b sy-ENVY
```

### link

```bash
chmod +x link.sh && ./link.sh
```
